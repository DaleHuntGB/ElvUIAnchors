## |cff1784d1ElvUI|r - EnhancedAnchors

This **AddOn** is not affiliated nor developed by |cff1784d1ElvUI|r, it merely acts an improvement on the anchoring system by allowing users to anchor certain elements to any frames.

Additionally, this **AddOn** focuses on doing this in the most performant way possible.

### Important Notice
This will adjust the `E.db.movers` value in your existing |cff1784d1ElvUI|r Profiles. Unfortunately, for now, this is the only way this can be handled. This means that if the **AddOn** is uninstalled or disabled, the existing anchors will remain until it is moved manually via the |cff1784d1ElvUI|r **Movers** environment.
